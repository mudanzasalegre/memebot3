# Reports Regeneration

Core reports are regenerated by:

```powershell
.\.venv\Scripts\python.exe tools\regenerate_core_reports.py
```

The command is paper-safe. It creates valid JSON placeholders when source data is empty or a generator cannot produce rows, and each required report includes `generated_at_utc`.

Automatic regeneration is controlled by:

- `CORE_REPORTS_AUTO_REGEN_ENABLED=true`
- `CORE_REPORTS_REGEN_INTERVAL_MIN=30`
- `CORE_REPORTS_REGEN_ON_STARTUP=true`
- `CORE_REPORTS_REGEN_ON_CLOSES=25`

Required reports:

- `data/metrics/trade_diagnostics.json`
- `data/metrics/policy_replay.json`
- `data/metrics/missed_pumps.json`
- `data/metrics/post_hotfix_strategy_preview.json`
- `data/metrics/runner_capture_ladder_report.json`
- `data/metrics/untagged_buy_block_report.json`
- `data/metrics/sniper_research_subprofile_report.json`
- `data/metrics/runner_turbo_monitor_report.json`
